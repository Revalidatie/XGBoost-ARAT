Welcome to the XGBoost ARAT prediction tool

Welcome to web application developed to showcase an XGBoost prediction model of upper limb recovery of stroke patients. The app makes a prediction of the ARAT score at 6 months of an individual patient based on clinical data that you input.

This tool is currently a research tool and not yet suitable for clinical use.

Steps to get a prediction

  1. Upload patient data
    2. Predictions can be retrieved by uploading a file with comma-separated patient data in the panel on the left. This file should contain one or multiple measurements of ARAT, shoulder abduction, finger extension and the corresponding time of measurement of a single patient. A detailed description of the input variables is given below. You can download example patient files in the left panel.
  2. Choose moment of model application
    3. After uploading is complete, a slider will appear that lets you select the moment in time from which you want to make the predictions. For example, if you choose 7 days post stroke, the model will predict as if it was applied at that time, with only the measurements that were taken in the first week.
  3. View prediction
    4. Predictions for your patient can be viewed in the tab View prediction. The prediction output will dynamically change when you select different time points with the slider.

Description of input data

Variable Description Range
ARAT Action Research Arm Test [0-57]
SA Fugl-Meyer Shoulder Abduction 0: no random movement
9: random activity palpable, no movement visible
14: random movement seeable but not seeable in total movement range
19: random movement across total movement range, not possible against resistance
25: random movement against resistance, but weaker than contralateral side
33: normal strength in comparison with contralateral side
